## Note for AI Communication:
- java script run faster than python when just calling API
- install node.js https://nodejs.org/en/download
- use `npm install` in backend folder to install package.json
- install Live Server extension https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer
- run `node ai.js` in terminal (`cd` to `backend/src`) then run Live Server in html file
